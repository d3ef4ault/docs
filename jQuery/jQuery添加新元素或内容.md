# jQuery添加新元素或内容

## append()

在被选元素**标签内的结尾**插入内容

```js
$("p").append("Some appended text.");
```

## prepend()

在被选元素**标签内的开头**插入内容

```js
$("p").prepend("Some prepended text.");
```

## after()

在被选元素**标签外之后**插入内容

```js
$("img").after("Some text after");
```

## before()

在被选元素**标签外之前**插入内容

```js
$("img").before("Some text before");
```
